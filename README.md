# Lottery2
##  C#抽奖程序，主要用于公司年会时的抽奖活动，共分两个版本(1.[文字版](https://github.com/weishaoying/Lottery) 2.头像版)这里是头像版，具有如下功能：<br/>
开发环境：VS2010 +.Net Framework4.0
<br/>
1. 自定义人员图片，初始化奖池<br/>
2. 奖金等级分为特等奖、一等奖、二等奖、三等奖、纪念奖等<br/>
3. 点击抽奖后头像快速循环滚动<br/>
4. 可以调整滚动速度<br/>
